# Cripto-7
Repositorio para el código de el trabajo de criptografía

## Instalación y ejecución de función principal
```
- pip install -r requirements.txt
- cd video
- python3 c-video.py
```

## Instalación y ejecución de tests
```
- pip install -r requirements.txt
- sudo apt-get install python3-tk
- cd tests
- python3 testx.py
```
